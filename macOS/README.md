# Kids Schedule Studio For macOS

This folder is for MacBook and iMac users.

## Fastest Way To Run

1. Download or clone the GitHub repo.
2. Open the `macOS` folder.
3. Double-click:

```text
Run Kids Schedule Studio.command
```

4. Your browser should open automatically at:

```text
http://127.0.0.1:4177/index.html
```

Keep the Terminal window open while using the app. Closing that Terminal window stops the local app server.

## First Launch Security Prompt

If macOS blocks the command file the first time:

1. Right-click `Run Kids Schedule Studio.command`.
2. Choose Open.
3. Confirm Open.

## Build A Mac Desktop App

If you want a real `.app` bundle with the Kids Schedule Studio icon:

1. Open the `macOS` folder.
2. Double-click:

```text
Build Mac Desktop App.command
```

The generated app is created here:

```text
dist/Kids Schedule Studio.app
```

The generated zip is created here:

```text
dist/Kids Schedule Studio.zip
```

## Requirement

Python 3 must be installed. If the launcher says Python 3 is missing, install it from:

```text
https://www.python.org/downloads/macos/
```

## Saving

The app auto-saves locally in your browser. Use Save Now for manual confirmation. Use Export backup after major edits for extra safety.
