# Kids Schedule Studio For Windows

This folder is for Windows users.

## Fastest Way To Run

1. Download or clone the GitHub repo.
2. Extract the zip if you downloaded one.
3. Open the `Windows` folder.
4. Double-click:

```text
Run Kids Schedule Studio.bat
```

5. Your browser should open automatically at:

```text
http://127.0.0.1:4177/index.html
```

Keep the command window open while using the app. Closing that window stops the local app server.

## If Windows Asks For Permission

The launcher only starts a local server on your own computer at `127.0.0.1`.

If Windows SmartScreen asks, choose More info, then Run anyway only if you downloaded the app from the GitHub repo or release you trust.

## Requirement

Python 3 must be installed. The launcher checks for Python automatically.

If Python is missing, install it from:

```text
https://www.python.org/downloads/windows/
```

During install, check:

```text
Add python.exe to PATH
```

Then double-click `Run Kids Schedule Studio.bat` again.

## Printing PDF

Use Print Week PDF, Print Day PDF, or Print Month PDF inside the app. In the browser print window, choose Save as PDF.

## Saving

The app auto-saves locally in your browser. Use Save Now for manual confirmation. Use Export backup after major edits for extra safety.
